51.52. pg_statistic_ext_data

目录pg_statistic_ext_data保存在pg_statistic_ext中定义的扩展规划器统计信息的数据。该目录的每一行对应用CREATE STATISTICS创建的一个统计信息对象。

就像pg_statistic，pg_statistic_ext_data不应该被公众读取到，因为其内容可能被视为敏感内容。（例如：列中值的最常见组合可能非常令人关注。）pg_stats_ext是pg_statistic_ext_data上的公共可读视图（与pg_statistic_ext连接后），它仅暴露有关当前用户可读的表和列的信息。
