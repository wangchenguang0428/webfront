51.58. pg_ts_config

pg_ts_config目录包含表示文本搜索配置的选项。一个配置指定了一个特定的文本搜索分析器和一个用于分析器输出记号的字典列表。分析器由pg_ts_config项展现，而记号到字典的映射则由pg_ts_config_map中的辅助项定义。

PostgreSQL的文本搜索特性在第 12 章中有更详尽的描述。
