51.6. pg_attrdef

pg_attrdef存储列的默认值。列的主要信息存储在pg_attribute。只有那些显式指定了一个默认值的列才会在这个目录中有一个项。

表 51.6. pg_attrdef的列

名称类型引用描述
oidoid 行标识符
adrelidoidpg_class.oid该列所属的表
adnumint2pg_attribute.attnum列号
adbinpg_node_tree 列的默认值，以nodeToString()表示。用pg_get_expr(adbin, adrelid)将其转换为SQL表达式。
