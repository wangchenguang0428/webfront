SPI_getargtypeid

SPI_getargtypeid — 为由SPI_prepare 准备好的一个语句的一个参数返回其数据类型 OID

大纲

Oid SPI_getargtypeid(SPIPlanPtr plan, int argIndex)

描述

SPI_getargtypeid返回由 SPI_prepare准备好的一个语句的 第argIndex个参数的类型的 OID。 第一个参数的索引为零。

参数

SPIPlanPtr plan

预备语句（由SPI_prepare返回）

int argIndex

参数的索引，从零开始

返回值

给定索引处的参数的类型 OID。如果该 plan为NULL或者无效， 或者argIndex小于零或者小于为 plan声明的参数数量， SPI_result会被设置为 SPI_ERROR_ARGUMENT并且将会返回 InvalidOid。
