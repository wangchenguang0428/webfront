51.80. pg_replication_origin_status

pg_replication_origin_status视图包含有关一个特定源已经重放了多少的信息。更多有关复制源的内容请见第 49 章。

表 51.81. pg_replication_origin_status的列

名称类型引用描述
local_idoidpg_replication_origin.roident内部的节点标识符
external_idtextpg_replication_origin.roname外部的节点标识符
remote_lsnpg_lsn 源节点的 LSN，到这个位置的数据都已经被复制。
local_lsnpg_lsn  这个节点的 LSN，remote_lsn已经被复制到这里。使用异步提交时，在将数据持久化到磁盘前用它来刷入提交记录。
