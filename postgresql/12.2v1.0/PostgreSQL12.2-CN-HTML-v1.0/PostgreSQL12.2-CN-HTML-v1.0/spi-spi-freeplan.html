SPI_freeplan

SPI_freeplan — 释放一个之前保存的预备语句

大纲

int SPI_freeplan(SPIPlanPtr plan)

描述

SPI_freeplan释放一个之前由 SPI_prepare返回的或者由 SPI_keepplan、SPI_saveplan 保存的预备语句。

参数

SPIPlanPtr plan

要释放的语句的指针

返回值

成功返回 0；如果plan为 NULL或无效则返回 SPI_ERROR_ARGUMENT
