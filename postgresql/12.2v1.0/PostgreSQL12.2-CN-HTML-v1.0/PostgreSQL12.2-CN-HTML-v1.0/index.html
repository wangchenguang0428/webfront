摘要

PostgreSQL 9.6版本以后的中文手册最初基于彭煜玮副教授翻译的 《PostgreSQL 9.6.0 文档》，后续版本的中文手册主要在前一版本的基础上作增量翻译。 《PostgreSQL 12.2手册》基于前一版本的《PostgreSQL 11.2手册》翻译。 翻译工作由文档翻译组志愿者chegong18,sizhitu,sunshinerxu,ChenHuajun,jingsam,messon007等完成。 详细请参考PostgreSQL 12中文手册的翻译。 如果发现中文手册中存在问题，请向Github源码仓库提交问题报告(Issue)或修订Patch(PR)。

中文手册版本:1.0 最后更新时间:2020-07-25
