65.2. 内建操作符类

表 65.1中展示了PostgreSQL 核心发布所包括的SP-GiST操作符类。

表 65.1. 内建 SP-GiST 操作符类

名称索引数据类型可索引操作符
kd_point_opspoint << <@ <^ >> >^ ~=
quad_point_opspoint << <@ <^ >> >^ ~=
range_ops任何范围类型 && &< &> -|- << <@ = >> @>
box_opsbox << &< && &> >> ~= @> <@ &<| <<| |>> |&>
poly_opspolygon << &< && &> >> ~= @> <@ &<| <<| |>> |&>
text_opstext < <= = > >= ~<=~ ~<~ ~>=~ ~>~ ^@
inet_opsinet, cidr && >> >>= > >= <> << <<= < <= =

在用于类型point的两种操作符类中，quad_point_ops是默认值。kd_point_ops支持相同的操作符，但是使用一种不同的索引数据结构，在某些应用中它可以提供更好的性能。
