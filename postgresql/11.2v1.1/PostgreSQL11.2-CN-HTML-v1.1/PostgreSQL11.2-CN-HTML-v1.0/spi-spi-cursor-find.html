SPI_cursor_find

SPI_cursor_find — 用名称查找一个现有的游标

大纲

Portal SPI_cursor_find(const char * name)

描述

SPI_cursor_find用名称查找一个现有的 portal。 这主要被用于解析由其他某个函数返回的一个油表名称。

参数

const char * name

该 portal 的名称

返回值

带有指定名称的 portal 的指针，如果没有找到就是 NULL
