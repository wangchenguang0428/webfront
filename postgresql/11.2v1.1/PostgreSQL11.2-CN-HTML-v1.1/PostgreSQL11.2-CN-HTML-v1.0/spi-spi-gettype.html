SPI_gettype

SPI_gettype — 返回指定列的数据类型名称

大纲

char * SPI_gettype(TupleDesc rowdesc, int colnumber)

描述

SPI_gettype返回该指定列的数据类型名称 的拷贝（当你不再需要该拷贝后，可以使用pfree 释放它）。

参数

TupleDesc rowdesc

输入行描述

int colnumber

列号（从 1 开始计）

返回值

指定列的数据类型名称，或者在错误时返回NULL。 错误时SPI_result会被设置成 SPI_ERROR_NOATTRIBUTE。
