SPI_getargcount

SPI_getargcount — 返回一个由SPI_prepare 准备好的语句所需的参数数量

大纲

int SPI_getargcount(SPIPlanPtr plan)

描述

SPI_getargcount返回执行一个由 SPI_prepare准备好的语句所需的参数数量。

参数

SPIPlanPtr plan

预备语句（由SPI_prepare返回）

返回值

plan所期望的参数计数。如果该 plan为NULL或者无效， SPI_result会被设置为SPI_ERROR_ARGUMENT 并且返回 -1。
